# Tamper-Evident Audit Log (One-Hour Stack)
FastAPI + SQLite (append-only hash chain) + simple RBAC + masking exports + access reports. Tiny React UI.

## Quick start
- Install Docker.
```bash
docker compose up --build
```
- UI: http://localhost:5173
- API docs: http://localhost:8000/docs

Demo API keys:
- admin: ADMIN_DEMO_KEY
- auditor: AUDITOR_DEMO_KEY
- investigator: INVESTIGATOR_DEMO_KEY
